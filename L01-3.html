基礎編　03.関数

自分で機能を作る

同じような処理をプログラムのあちこちで行う場合、あらかじめ一連の処理を一つの機能としてひとまとめにしておくことで、後から簡単にその処理を呼び出して実行させることができます。ひとまとめにした機能のことを関数と呼びます。

関数という用語も数学用語から輸入されたものです。英語ではFunction、つまり何らかの機能や働きといった意味です。数学界の「関数」は、「ある変数に依存して決まる値」「入力値と出力値の対応を表す式」というような意味ですが、プログラミング界では必ずしも入力に対して数学のように何かしらの出力がある訳ではないので、もっと漠然と広い意味で「何らかの機能や働き」くらいの理解でよいかと思います。

関数の定義

関数を定義し、呼び出すには、次のように記述します。 

        # 関数の定義
    def 関数名(引数1, 引数2, ...):
        関数化したい処理
        return 返したい値1, 返したい値2,... #省略可

    # 関数の呼び出し
    # 関数が何の値も返さないとき
    関数名(引数1, 引数2, ...)
    # 関数から返ってくる値があるとき
    変数名 = 関数名(引数1, 引数2, ...)
    # returnが複数ある場合はタプル（要素の変更ができないリスト）として返ってくる

    # 具体例：2から指定した数字までの素数を調べるプログラム
    def isPrime(number):
        counter = 0 #割り切れた回数を数えるカウンター
        # 2〜N-1で割り切れる個数を数える
        for i in range(2, number):
            if number % i == 0:
                counter = counter + 1  #割り切れたらcounterに１を追加
        if counter == 0: #素数ならその値を表示し、値1を返す
            print(f"{number},")
            return 1
        else:  #素数でなければ値０を返す
            return 0

    #ここからメインの処理
    inputNum = int(input("２以上の自然数を入力："))
    countPrime = 0  #個数を数えるための変数

    for num in range(2, inputNum+1):
        countPrime = countPrime + isPrime(num)

    print(f"個数は{countPrime}です")

forやifのときと同様、コロンやインデントを忘れないでください。

他にも次のような注意点があります。

  • 関数の定義はプログラムのはじめ（ライブラリをインポートした後）にまとめて行うようにしましょう。
  • 関数名は基本的には自分の好きな名前をつけられますが、Pythonにデフォルトで備わっている機能（printなど）と同じ名前はつけられません。
  • 関数の中で使用される変数は、関数の外から呼び出すことはできません。
