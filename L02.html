ライブラリの活用

便利機能のセット

プログラミングに慣れてくると、ただの数値計算だけではなく、もっと複雑なことを処理させたくなります。しかし、複雑なことを実現するプログラムを自分でイチから作成するのは、ハードルが高いし面倒ですよね。それに、大抵の場合、自分がやりたいと思っているようなことは既に他の人も同じように考えていることが多いです。そして、偉大な先人が我々のようなプログラミングの後輩たちのために、複雑な処理を簡単に実現できるような道具を揃えておいてくれています。この道具のことを、一般的にライブラリと呼んでいます

皆さんがよく使いそうなライブラリを挙げます。

  • math：三角関数や円周率など、数学に特化したもの
  • numpy：多次元配列を用いた演算
  • pandas：統計量の表示やグラフ化
  • datetime：日付・時刻の取り扱い
  • os：ディレクトリやファイル操作
  • arcpy：ArcGISの機能を使用
  • pyautogui：GUI操作の自動化
  • python-pptx：pptx（パワポ）ファイルの操作・編集

使い方

どのライブラリであっても、最初にやることはインポートです。通常、プログラムの最初に行います。インポートの方法はいくつかありますが、大きく分けて、ライブラリの機能全てをインポートする方法と、ライブラリの機能の中で特定のものを指定してインポートする方法の2つがあります。

インポートした後は、プログラム中で（ライブラリ名）.（機能名）で呼び出します。

        # ライブラリのインポート
    import arcpy  # arcpyを普通にインポート
    import numpy as np  # numpyをインポートし、プログラム中ではnpで呼び出す
    import matplotlib.pyplot as plt  # matplotlibライブラリのpyplotという機能だけをインポートし、プログラム中ではpltとして呼び出す
    from matplotlib import pyplot as plt  # 上と同じ働きをする（別の書き方）

    # インポートしたライブラリを使う
    # 3x3で要素がすべて0の二次元配列を作成
    zeroArray = np.zeros((3, 3))

個々のライブラリの詳しい使い方は、ここではとても挙げきれないので必要に応じて調べてください。ライブラリ名と使いたい機能を検索すれば、使い方の例が出てくると思います。

便利機能を自分で作る

ときには自分で作ったたくさんの関数を、複数のプログラムで使用したいことがあると思います。このようなときには、モジュールを作成することで自作関数をライブラリのように呼び出すことができます。

モジュールは、自作した関数（defから始まるブロック）を書いたファイルを「.py」という拡張子をつけて、後でこのモジュールを呼び出すプログラムが保存されている場所と同じところに保存することで作成できます。プログラムでインポートするには、import モジュール名（拡張子なし）と書きます。

          # モジュールのインポート
      # myutil.pyというモジュールを作成したとする
      import myutil  # myutilをインポート
      import myutil as ut  # 「as なんとか」の書き方はモジュールでも可能

      # インポートしたモジュールのsampleという機能を使う
      a = ut.sample()
